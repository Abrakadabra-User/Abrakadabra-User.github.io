### Core Technologies & Contributions
This project aims to translate cutting-edge Underwater Image Enhancement (UIE) theories into engineering applications that address real-world pain points in aquatic detection. As the first author, this core research was officially accepted by the JCR Q1 journal *Remote Sensing* in May 2026.

* **Algorithm Architecture Design for Real-World Demands**: Jointly funded and supported by multiple provincial and ministerial key sci-tech projects, this work focuses on the challenge of visual enhancement in complex degraded underwater scenes, providing foundational visual algorithm support for actual navigational and transportation underwater detection tasks.
* **Fusion of Physical Priors and State Space Models**: Based on the Python and PyTorch frameworks, successfully bridged the deep integration of traditional underwater imaging physical priors with State Space Models (Mamba architecture) at the code level, innovatively constructing the Scene-Aware Retinex Mamba (SARM) network model.
* **Efficient Model Training and Backpropagation Tuning**: Designed and implemented highly efficient feature extraction modules and customized loss functions tailored for highly complex real-world underwater degraded data. By fine-tuning the network, the model's convergence efficiency and robustness during forward propagation and backward gradient updates were significantly improved.
