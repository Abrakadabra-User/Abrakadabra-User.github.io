这里展示了我在本科阶段参与的核心科研课题与工程落地项目。我的研究不仅关注算法的理论创新，更致力于跨越理论与真实物理世界之间的鸿沟，强调整体系统架构与底层代码实现。

### 项目概览

- **物联感知信息驱动的历史文化街区活力动态仿真推演**：国家级重点课题子课题。基于 Unreal Engine (UE) 与 Blender，构建与真实世界时间、天气高度同步的高保真虚拟推演三维底座。
- **基于 RF-LSTM 的幕阜山森林生态系统土壤呼吸研究**：“探索与猜想计划”重点项目。负责构建机理与深度学习耦合模型，将生态学先验知识融入算法以提升预测精度与可解释性。
- **基于 RAG 的 API 检索引擎开发**：面向工业界真实落地需求，研发基于检索增强生成 (RAG) 架构的大语言模型智能检索系统。
- **SARM 视觉增强框架**：顶刊学术成果工程化。基于 Python/PyTorch 框架实现水下图像增强网络，打通物理先验与状态空间模型 (Mamba) 的深度整合。
- **卫星星座空间计算与多任务规划调度引擎**：底层空间计算引擎研发。基于球面网格分治算法与动态规划，实现多星对地覆盖的高精度仿真与观测任务最优调度。
